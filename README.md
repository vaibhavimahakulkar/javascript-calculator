# JavaScript Calculator

A simple console-based calculator built using JavaScript to practice fundamental programming concepts.

## Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🚫 Division by zero validation
- 🔁 Perform multiple calculations in one session

## Concepts Practiced

- Variables
- Data Types
- Type Conversion
- Input & Output
- Conditional Statements
- Loops
- Switch Statement

## Technologies Used

- HTML5
- JavaScript (ES6)
